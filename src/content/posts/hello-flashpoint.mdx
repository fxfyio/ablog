这是一篇用于初始化博客结构的样例文章。

我希望这个站点保持两个特性：

1. 内容文件在仓库里，可长期迁移。
2. 写作体验足够顺手，手机和电脑都能完成。

## 写作与发布

当前方案是 `Markdown/MDX + Keystatic`：

- 想直接写时，用编辑器维护 `src/content/posts`
- 想快速改稿时，用 `/keystatic` 后台

## 一个代码片段

```ts
type Flash = {
  title: string;
  intensity: number;
};

export function forge(input: Flash) {
  return `${input.title} · ${input.intensity}`;
}
```
